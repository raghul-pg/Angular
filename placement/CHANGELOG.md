# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-20

### Added
- Initial release of Campus Placement Portal
- CGPA-based company filtering
- Branch-based filtering
- Expandable company cards with detailed information
- Company salary breakdown display
- Interview process details
- Selection rounds information
- Responsive design for all devices
- Modern teal/cyan color scheme
- Smooth animations and transitions
- Real-time filtering
- Sample data for 6 companies (Google, Microsoft, Meta, Amazon, TCS, Infosys)

### Features
- Filter companies by student CGPA
- Filter by branch of study
- View comprehensive company information
- Expandable/collapsible cards
- Company website links
- Reset filters functionality
- Results counter
- Mobile-responsive design

### Architecture
- Angular 19 framework
- TypeScript for type safety
- Service-based architecture
- Model-based data structure
- Component-based UI
- CSS3 styling with gradients
- Reactive forms with RxJS

### Documentation
- Comprehensive README.md
- Contributing guidelines
- License (MIT)
- Installation and setup instructions
- Usage guide
- Future enhancements roadmap
